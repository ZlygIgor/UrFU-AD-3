## Цель работы
Разработать оптимальный баланс для десяти уровней игры Dragon Picker

## Задание 1
### Предложите вариант изменения найденных переменных для 10 уровней в игре. Визуализируйте изменение уровня сложности в таблице. 
Ход работы:
- Скачайть и проанализировать прототип игры Dragon Picker и выделить переменные отвечающие за перемещение дракона и сброс яиц
- Проанализировав данный прототип можем выделать следущие переменные:

- speed - Скорость перемещения дракона
- timeBetweenEggDrops - Интервал сброса яиц
- leftRightDistance - Переменная отображающая то насколько дракон может перемещаться в стороны
- chanceDirection - Шанс того что дракон сменит направление перемещения
- myVector - Вектор перемещения яйца

Для изменения уровня сложности можно изменять скорость перемещения дракона и количество сбрасываемых им яиц в секунду:
(https://docs.google.com/spreadsheets/d/1-GkqI24Acz1TX7VuRFdhTeJjZLO5Js0lyxzMNHD_s9g/edit?usp=sharing)
